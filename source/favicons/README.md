# Папка для фавиконок

Здесь должны находиться все необходимые варианты фавиконок, кроме `favicon.ico`, который должен быть в `source/`.

```shell
└── source/
    └── favicons/
        ├── android-chrome-192x192.png
        ├── android-chrome-512x512.png
        ├── apple-touch-icon.png
        ├── favicon-16x16.png
        ├── favicon-32x32.png
        ├── favicon-dark.png              # 32х32 для темной темы
        ├── favicon-light.png             # 32х32 для светлой темы
        ├── icon.svg                      # внутри две иконки (dark и light)
        ├── mstile-70x70.png              # на прозрачном фоне
        ├── mstile-144x144.png            # на прозрачном фоне
        ├── mstile-150x150.png            # на прозрачном фоне
        ├── mstile-310x150.png            # на прозрачном фоне
        ├── mstile-310x310.png            # на прозрачном фоне
        ├── safari-pinned-tab.svg         # маска для маков/сафари
        └── browserconfig.xml             # конфиг для браузера
```
